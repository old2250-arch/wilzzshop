Get Price List

GET /h2h/price-list

Retrieve the list of available products and their prices. Can be filtered by provider.

Query Parameters

Name Type Required Description
apikey string Yes Your API Key.
provider string No Filter results by specific provider (e.g., DANA, TELKOMSEL). If omitted, returns all products.

Response (200 OK)

Response (401 Unauthorized)

Response (200 OK - No products found for provider)

Orders

Create New Order

GET Note: POST is generally preferred for creating resources, but your API uses GET. /h2h/order/create

Place a new order for a specific product using its code and target.

Query Parameters

Name Type Required Description
code string Yes The product code (e.g., ML3, DANA1) from the Price List.
target string Yes The target identifier for the product (e.g., phone number for pulsa, User ID|Zone ID for games).
apikey string Yes Your API Key.

Response (202 Accepted)

Response (400 Bad Request / 401 Unauthorized / 500 Internal Server Error)

Cancel Deposit Request

GET Note: DELETE or POST is generally preferred for modifying resources, but your API uses GET. /h2h/deposit/cancel

Cancel a pending deposit request using its transaction ID.

Note: Cancellation is usually only possible for deposits that are still in 'pending' status and have not been paid.

Query Parameters

Name Type Required Description
trxid string Yes The deposit transaction ID (id field from the create deposit response) to cancel.
apikey string Yes Your API Key.

Response (200 OK)

Response (400 Bad Request / 401 Unauthorized / 404 Not Found / 409 Conflict)
