Pengantar

Selamat datang di dokumentasi API resmi layanan kami. API ini menyediakan antarmuka terprogram untuk mengakses berbagai fitur dan layanan yang kami sediakan.

Dokumentasi ini menjelaskan cara menggunakan API kami dengan contoh permintaan dan respons dalam format JSON.

Base URL 

https://www.v-pedia.web.id

Endpoint Order

Request 

GET /h2h/order/create?code=CODE-PRODUK&target=NO-TUJUAN&apikey=YOUR-APIKEY

Endpoint Buy Panel

Membeli paket panel dengan username dan paket RAM tertentu. Paket tersedia: 1gb, 2gb, 3gb, 4gb, 5gb, 6gb, 7gb, 8gb, 9gb, 10gb, unli.

Request 

/api/v1/cpanel?username=USERNAME-PANEL&paket=PAKET-RAM&apikey=YOUR-APIKEY

Response Success 

{
  "status": true,
  "message": "User dan server berhasil dibuat!",
  "user": {
    "id": 20,
    "username": "XXXXXXXXXXXXXXX",
    "email": "XXXXXXXXXXXXXXX@gmail.com",

  },
  "server": {
    "id": 22,
    "name": "XXXXXXXXXXXXXXX",
    "memory": 1015,

Contoh Response Error

Berikut adalah contoh respons error yang mungkin Anda temui saat menggunakan API.

APIKEY HARUS DI ISI

{
  "success": false,
  "message": "Parameter \"apikey\" harus diisi"
}

AKUN BELUM TERVERIFIKASI

{
  "success": false,
  "message": "Akun belum terverifikasi. Silakan verifikasi akun terlebih dahulu."
}

REQUEST TERLALU CEPAT

{
  "success": false,
  "message": "Anda harus menunggu 5 detik sebelum melakukan request lagi."
}

SERVER ERROR

{
  "success": false,
  "message": "Internal server error"
}

SALDO TIDAK CUKUP

{
  "success": false,
  "message": "Saldo tidak mencukupi"
}
